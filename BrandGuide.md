# Nebraskache Brand Guide

### Colors

- Primary: Deep Prairie Green
  - HEX: #1F6F43
  - Tints: 75% #579381 | 50% #8FB7A1 | 25% #C7DBD0

- Secondary: Prairie Gold
  - HEX: #E2B23A
  - Tints: 75% #EBC86B | 50% #F1D99C | 25% #F8EDCE

- Neutrals:
  - Prairie Mist (Background) — #F4F7F5
  - Compass Gray (Primary Text) — #333333
  - Trail Gray (Secondary Text) — #666666

- Usage
    - Backgrounds: Sandstone White or 25% tints
    - CTAs / Links: Sky Blue
    - Highlights: Prairie Gold
    - Text: Compass Gray

### Typography

- Headings: Montserrat
  - Use for titles, nav menus, section headers.

- Body: Lato
  - Use for all paragraphs, descriptions, small UI text.

- Accent: Bitter
  - Use sparingly: pull quotes, event posters, stickers, or taglines.

- Hierarchy Example:
    - H1 (Montserrat Bold, Gold)
    - H2 (Montserrat SemiBold, Blue)
    - Body (Lato Regular, Gray)
    - Accent (Bitter Bold, Gold or Blue)

### Tone & Assets

- Voice: Welcoming, adventurous, community-driven.
- Textures: Subtle topo lines, compass grids, trail paths as background patterns.
- Icons: Rounded line icons (Feather or Heroicons work well) in Sky Blue or Compass Gray.
- Logo: `/nebraskache.svg` or `/nebraskache-sans.svg` for a scalable, horizontal logo with transparent background.
  - `/nebraskache.svg` includes the text "NEBRASKACHE" under the main logo.
  - `/nebraskache-sans.svg` is only the main logo, without the text. Prefer this version when accompanied by an HTML element with the word "Nebraskache", such as in the header or footer.
